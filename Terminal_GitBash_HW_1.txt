1) Посмотреть где я -
pwd

2) Создать папку -
mkdir foldername

3) Зайти в папку -
cd foldername

4) Создать 3 папки -
mkdir A_1 B_2 C_3

5) Зайти в любую папку -
cd A_1

6) Создать 5 файлов (3 txt, 2 json) -
touch a_1.txt b_2.txt c_3.txt d_4.json e_5.json

7) Создать 3 папки -
mkdir D_4 E_5 F_6

8) Вывести список содержимого папки -
ls -la

9) + открыть любой txt файл -
cat a_1.txt

10) + Написать туда что-нибудь, любой текст -
cat >> a_1.txt, вводим текст peace to world, нажимаем кнопку enter

11) + сохранить и выйти -
Ctrl-C

12) Выйти из папки на уровень выше -
cd ..

__

13) Переместить любые 2 файла, которые вы создали, в любую другую папку -
mv A_1/{a_1.txt,d_4.json} B_2

14) Скопировать любые 2 файла, которые вы создали, в любую другую папку -
cp A_1/{b_2.txt,e_5.json} C_3

15) Найти файл по имени -
find . -name "c_3.txt"

16) Посмотреть содержимое в реальном времени (команда grep). Изучите как она работает -
tail -f A_1/c_3.txt
Выйти из режима просмотра - Ctrl-C. По умолчанию отражает только последние 10 строк. Если смотреть в реальном времени только
последние несколько строк файла, то необходимо использовать флаг -n в сочетании с флагом -f
tail -n3 -f A_1/c_3.txt. Выйти - Ctrl-C. Команда grep для просмотра содержимого в реальном времени
не используется, в основном используется для поиска.

17) Вывести несколько первых строк из текстового файла -
head -n3 A_1/c_3.txt

18) Вывести несколько последних строк из текстового файла -
tail -n3 A_1/c_3.txt

19) Просмотреть содержимое длинного файла (команда less). Изучите как она работает -
less A_1/c_3.txt
Выйти из режима просмотра - q

20) Вывести дату и время -
date +"%D %T"

http://162.55.220.72:5005/terminal-hw-request

		curl "http://162.55.220.72:5005/get_method?name=S&age=400"
с ответом ["S", "400"]

2) Написать скрипт, который выполнит автоматически пункты 3, 4, 5, 6, 7, 8, 13
- Командой touch создаем файл myscript
- в терминале вызываем текстовый редактор с именим файла (vim myscript), входим в режим редактирования (клавиша i)
- в первой строке указываем оболочку #! /bin/bash, переходим на другую строку (клавиша enter)
- в последующих строках вводим по одной команде из пунктов 3, 4, 5, 6, 7, 8, 13
(cd foldername
mkdir A_1 B_2 C_3
cd A_1
touch a_1.txt b_2.txt c_3.txt d_4.json e_5.json
mkdir D_4 E_5 F_6
ls -la
mv A_1/{a_1.txt,d_4.json} B_2
)
нажимаем клавишу enter, выходим из режима редактирования (клавиша ESC),
измененяем и закрываем файл (:wq) и нажимаем клавишу enter
- для того чтобы сделать файл испоняемым в терминале вводим chmod +x ./myscript
- выполняем файл ./myscript
